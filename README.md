# 🏪 BCcard & E-mart24 Data Analysis Project

BC카드와 이마트 병합 데이터를 활용하여 **MZ세대의 소비 패턴 분석 및 맞춤형 프로모션 전략**을 제안한 프로젝트입니다.  
데이터 기반의 인사이트를 통해 실제 매출 향상에 기여할 수 있는 방안을 탐색했습니다.

## 💡 주제 선정 이유

최근 경제 불황이 지속되며 물가 상승에 대한 부담감이 커지고 있는 상태입니다.  
이러한 환경 속에서 MZ세대는 “**요노(You Only Need One)**”라는 새로운 소비 트렌드를 형성하고 있습니다.  
요노는 미니멀리즘과 제로웨이스트가 결합된 개념으로, 불필요한 소비를 줄이고 자신에게 꼭 필요한 것에만 집중하는 가치관을 반영합니다.  

더불어 “**무지출 챌린지**”까지 확산되며 MZ세대의 지갑을 열기 어려워진 상황입니다.  

그래서 저는 이런 환경 속에서 **BC카드와 이마트24 결제 데이터를 분석**하여,  
MZ세대가 실제로 어떤 품목을 소비하고 있으며,  
그들의 **행동 패턴을 기반으로 효과적인 매출 증대 프로모션을 제시**하고자 했습니다.
또한 구매 금액에 가장 많은 영향을 주는 변수를 찾는 **회귀 예측 모델**을 제작하였습니다

GS25 통계에 따르면 편의점의 잡곡 매출은  
2022년 15.4%, 2023년 23.8%, 2024년 25.9%로 매년 증가하고 있습니다.  
또한 코로나19 이후 자극적인 음식에 질린 소비자들이 ‘간편한 건강식’으로 관심을 돌리고 있습니다.

이러한 트렌드와 소비 패턴 분석 결과를 결합해  
평일 오후 간식 시간대(18~20시)에 다음과 같은 프로모션을 제안합니다:

> 💡 **프로모션 아이디어:**
